Below are commands for setup he github in local system

        Step 1 : Clone
            git clone https://github.com/techrohit13/gitdemo.git

        Step 2 : Status
            git status  
        
        Step 3 : Add
            git add . or git add index.html
        
        Step 4 : Commit
            git commit -m "Index File added "
        
        Step 5 : Push to github
            git push -u origin main
